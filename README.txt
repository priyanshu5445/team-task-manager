## Description
TaskFlow is a full-stack MERN application designed for seamless team collaboration and task management. It features a modern, responsive UI with dark mode aesthetics, role-based access control (Admin/Member), and real-time dashboard statistics.

## Tech Stack
- Frontend: React, Vite, Vanilla CSS
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Deployment: Railway
